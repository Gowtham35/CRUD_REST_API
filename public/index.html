Welcome to my project where you can perform CRUD operations


Available routes:

GET request "/users" --- list all the users in the users table

GET request "/users/:id" --- get a individual user data from the users table

POST request "/users" --- creates new record in users table, pass the {id, name, age, gender} in body

PATCH request "/users/:id" --- updates the record in users table based on id you have sent in params, pass the {name, age, gender} in body thate you want to update

DELETE request "/users/:id" --- deletes the record in users table based on id you have sent in params


Note:

Use "boomerang"(a chrome extension) or "postman" to make the request

I have used postgres sql as db so make sure you have installed postgres in your system and make changes in db.js

while making a req from body use raw json format
